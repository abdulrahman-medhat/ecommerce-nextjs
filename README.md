# 🛒 Modern E-Commerce Project

A **full-featured online shopping platform** built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **ShadCN UI**.  
Now with **Dark Mode** and a fully responsive design for a smooth user experience.

## 🚀 Features
- **Next.js & TypeScript** – fast, scalable, and type-safe frontend  
- **Tailwind CSS & ShadCN UI** – clean and modern design  
- **Dark Mode** – switch easily between light & dark themes  
- **Dynamic Routing & API Integration** – browse products, manage cart, and place orders  
- **Responsive Design** – works perfectly on desktop, tablet, and mobile  

## 💻 Tech Stack
Next.js | TypeScript | Tailwind CSS | ShadCN UI | Dark Mode
